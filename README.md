Travel Planner — Сайт
------------------------------------------------------
Описание проекта
Travel Planner — это веб-приложение для планирования путешествий, которое помогает пользователю подобрать город для поездки, узнать актуальную погоду, посмотреть достопримечательности и составить персональный план путешествия по дням.
------------------------------------------------------
Основной функционал
- Поиск города и отображение общей информации
- Получение погоды по выбранному городу (через API)
- Просмотр достопримечательностей
- Фильтрация мест по категориям (музеи, парки, еда)
- Планирование поездки по дням
- Сохранение плана путешествия
- Смена языка интерфейса (RU / EN)
- Сохранение данных пользователя между сессиями
------------------------------------------------------
Страницы проекта
1. Главная — ввод города и начало планирования
2. Погода и общая информация
3. Достопримечательности
4. Маршруты и список мест
5. План поездки по дням
6. Настройки и сохранённые поездки
------------------------------------------------------
Используемые технологии
- HTML— семантическая разметка
- CSS— адаптивная верстка (Flexbox, Grid)
- JavaScript— логика и интерактив
- Fetch API— работа с внешними API
- Async / Await— асинхронность
- LocalStorage— хранение данных пользователя
- GitHub— контроль версий
------------------------------------------------------
Используемые API
- OpenWeatherMap API — получение данных о погоде
- OpenTripMap API — получение информации о достопримечательностях
