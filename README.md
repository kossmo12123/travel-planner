Maply - Путеводитель по миру

Описание

Maply— веб-приложение для планирования путешествий с поиском мест, прогнозом погоды, построением маршрутов и планировщиком поездок.

---------------------------------------------------------------------------------------------------------
Основной функционал

- Погода— текущая погода и прогноз на 5 дней для любого города
- Достопримечательности — поиск мест с фильтрацией по категориям (музеи, памятники, храмы, парки)
- Маршруты — построение маршрутов кликом по карте с расчётом расстояния
- Планировщик — создание и управление поездками
- Форум — сообщество путешественников
- Темы — светлая и тёмная темы оформления
---------------------------------------------------------------------------------------------------------
Страницы проекта (9 шт)

1. Главная — интерактивная карта
2. Погода — прогноз погоды
3. Достопримечательности — поиск мест
4. Маршруты — построение маршрутов
5. Планировщик — управление поездками
6. Форум — сообщество
7. О проекте
8. Авторизация — вход/регистрация
9. Профиль — личный кабинет
---------------------------------------------------------------------------------------------------------
Используемые технологии

Frontend: HTML5, CSS3, JavaScript ES6+  
Библиотеки: Leaflet 1.9.4, Google Fonts  
Методологии: Адаптивная вёрстка, Flexbox, Grid, async/await

---------------------------------------------------------------------------------------------------------
Используемые API (4 шт)

1. WeatherAPI — прогноз погоды
2. Geoapify Places — поиск достопримечательностей
3. Geoapify Geocoding — преобразование адресов в координаты
4. Geoapify Routing — построение маршрутов
---------------------------------------------------------------------------------------------------------
LocalStorage

Сохраняются:
- Список поездок
- Посты форума  
- Маршруты
- Данные пользователя
- Настройки темы
---------------------------------------------------------------------------------------------------------
Ключевые особенности

- Асинхронность: все API запросы через async/await с обработкой ошибок
- Состояния: индикаторы загрузки и сообщения об ошибках
- Адаптивность: 3 брейкпоинта (mobile/tablet/desktop)
- Интерактивность: 8 JavaScript сценариев
